## 📖 About This Repository

This repository serves as a living companion to the survey paper: **"Federated Learning for Cybersecurity: A Comprehensive Review"**.

As centralized security architectures struggle with the massive scale of IoT devices and strict data privacy regulations (e.g., GDPR), **Federated Learning (FL)** provides a privacy-preserving path forward. This repository curates the most impactful research covering intrusion detection, malware analysis, threat intelligence sharing, and adversarial robustness within federated networks.
